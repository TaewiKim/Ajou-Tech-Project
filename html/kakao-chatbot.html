카카오톡 AI 챗봇 프로젝트

이 프로젝트는 카카오톡 채널 API와 AWS 서버리스, AI 챗봇(ChatGPT)을 결합해 나만의 챗봇을 직접 만들고 배포해보는 창의적인 실습형 프로그램입니다.

  • 카카오톡 챗봇 플로우 설계 및 자동 응답
  • ChatGPT 연동을 통한 대화형 서비스 구축
  • AWS Lambda + API Gateway 서버리스 구조
  • S3 · DynamoDB를 활용한 데이터 저장

우리의 챗봇 결과물

성경암송 챗봇 "꼬봇"

카카오톡에서 말씀을 입력하면 암송 퀴즈를 내주고, 일상적인 대화도 해주는 스마트한 챗봇

네비게이토 성경암송 채널
